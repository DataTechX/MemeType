# MemeBot

เราไม่ได้จัดเตรียมวิธีการติดตั้ง การใช้งาน และเอกสารใดๆ ในปัจจุบัน เนื่องจาก API นั้นไม่เสถียร รหัสที่มานี้อยู่ภายใต้มาตรฐาน[MIT](LICENSE)
ต้นฉบับโดย bu

## ติดตั้ง
สามารถใช้ `npm` แทน `yarn` แนะนำให้ใช้ `yarn`
```sh
$ yarn install
```
